# Version 1.2

**Released:** 2026-07-14
**Previous version:** v1.1

---

## What Changed

### Fix: Buyer Presentation Exit Values Now Match Exit Grid Badges

The "Net Profit (incl. held CF)" values in the Buyer Presentation now always show the exact same numbers as the "Sale Exit Value at Balloon (with CF)" and "Refi Cash-Out at Balloon (with CF)" badges in the Cash Flow card.

**Root cause:** `generateBuyerPresentation` was recomputing exit values independently from `recalc`, which could produce drift under certain deal configurations.

**Fix:** Introduced `_exitCache` — a global object populated by `recalc` at the balloon year with all precomputed exit values. `generateBuyerPresentation` now reads directly from this cache, guaranteeing consistency.

**Values now cached and shared:**
- `exitPropVal`, `exitSrBal`, `exitSellerOwed`
- `cumCF`, `saleNet`, `saleNetWithCF`, `refiNet`, `refiNetWithCF`
- `pfCumCF`, `pfSaleNet`, `pfSaleWithCF`, `pfRefiNet`, `pfRefiWithCF`
- `pfNOIatBalloon`, `totalDebtMo`, `balloonYr`

---

### Improvement: Buyer Presentation — Current NOI Path Now Leads

Reordered the Buyer Presentation output so the **Current NOI Path exit** always appears first, followed by the **Proforma NOI Path exit** (when proforma NOI is set).

**Before (v1.1):**
```
CASH TO CLOSE
PROFORMA NOI ANALYSIS  ← appeared before current NOI exit
EXIT PROFIT PROJECTION (Current NOI Path)
DISCLAIMER
```

**After (v1.2):**
```
CASH TO CLOSE
EXIT PROFIT PROJECTION (Current NOI Path)  ← base case first, always visible
PROFORMA NOI ANALYSIS + PROFORMA EXIT  ← upside scenario follows
DISCLAIMER
```

The Current NOI Path section now also shows:
- Day-1 NOI and growth rate arrow
- NOI at Balloon (when NOI growth % > 0)
- Total appreciation gain since acquisition as a %

---

### Update: LOI Timeline

Updated the TIMELINE section at the end of every LOI.

**Before (v1.1):**
```
TIMELINE:
Closing: Within 45-90 days of signed purchase and sales agreement.
```

**After (v1.2):**
```
TIMELINE:
Closing: Within 45-60 days of signed purchase and sales agreement.
Buyer contingency for loan and appraisal at agreed upon total purchase price
```

---

### Update: Admin Template — Senior Rate Updated to 8%

Admin template default for senior loan rate updated from 7% → 8% to reflect current market rates.

---

## Summary of Changes

| Area | Change |
|---|---|
| Buyer Presentation math | Exit values now read from `_exitCache` set by `recalc` — no drift |
| Buyer Presentation layout | Current NOI path exit leads; proforma exit follows |
| Current NOI exit detail | Day-1 NOI, growth arrow, NOI at balloon, appreciation % all shown |
| LOI Timeline | 45-60 days (was 45-90); loan + appraisal contingency added |
| Admin template | Sr rate default 7% → 8% |
